Nonnon Typing 


	typing trainer

	[ Mode ]

	you can choose by menu

	"Warm-Up Mode"

		you need to answer a displayed key

		+ same sequence

	"Trainer Mode"

		you need to answer a displayed key

		+ random sequence

	"Free Mode"

		you can type freely

		+ a typed key name is displayed


	[ Hints ]

	1 : remember home position

		"F" and "J"

	2 : remember offset from home position

		use "Warm-Up Mode"

	3 : remember Return and Backspace

		don't forget to move back to home position
