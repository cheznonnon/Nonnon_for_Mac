Nonnon Poker 


	System Menu

		Bet Mode

			bet 100 coins, and return when you win

		Endless Mode

			no bet, no game over

	Input

		Click

			Cards

				hold/keep or not

			Button(s)

				Bet Mode

					"Bet"  : bet and see result, double-up is possible
					"Next" : play the next game

				Endless Mode

					"Draw" : see result
					"Next" : play the next game
