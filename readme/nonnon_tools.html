Nonnon Tools 


	Zero Timestamp

		+ set 0:00:00 in modified time


	Remove Dot Files

		+ remove dot files (macOS system hidden files)

		[ Caution ]

		all files named like ".name" will be removed

			+ e.g. "...Hits" is applicable


	Pack/Unpack Icons

		Windows® Icons (.ICO) handler

		[ Pack ]

		+ make a multi-icon from a folder contains .ICO or .PNG

		[ Unpack ]

		+ extract icon files from a multi-icon


	ArcDrop

		Zip Maker/Extractor

		+ this makes a Windows® ZipFolder compatible .ZIP file

		+ auto-detection available

		[ Caution ]

		don't use for Zips made by macOS, no data will be extracted

		unzip will overwrite an already exist folder automatically


	Favorites2HTML

		a folder includes .URL or .WEBLOC will be one file (.HTML)

		.HTML will be a folder contains .WEBLOC


	Nonnon MIDI

		make a midi file (.MID) from a text script

		+ Finder cannot play output files
		+ a special app is needed, search AppStore


	16-bit .WAV

		convert old Windows compatible .WAV

		+ an older file will be overwritten
