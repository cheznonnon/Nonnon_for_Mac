Nonnon Freecell 


	Freecell Solitaire

	[ Rule ]

		[ Stock ]

			top-right area

			put the same suit cards from A(1) to K(13)

			+ you will win when all cards are in this area

		[ Freecell ]

			top-left area

			you can put any cards but 4 cards only

			+ Auto-Move sees this area

		[ Tableau ]

			bottom area

			you cannot stack the same suit cards

			you can stack different color cards

			+ reds   (hearts and diamonds)
			+ blacks (spades and clubs)

			+ like reds(K) → blacks(Q) → reds(J) → blacks(10)...

			+ any starting numbers available
			+ like blacks(4) → reds(3) → blacks(2)

	[ Input ]

		Drag And Drop

			move a card

		Double Click

			auto-move to the Stock

		Auto-Move

			automatically move to the Stock when safely movable

	[ Titlebar Menu ]

		[ Replay ]

			undo the current session

		[ New Game ]

			start a new game
