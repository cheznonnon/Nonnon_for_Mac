Nonnon Nyaurism 


	.WAV Tweaker

		drop to load

		[ Supported Format ]

			+ .WAV
			+ some .MP3 or .M4A files are available (load only)


	Limitation

		full-scalable but suit for small sound effect files

			+ maybe too much heavy with large files

		no seed generation

			+ you can use your smartphone or MIDI recorder app to make seed

		window (plotter area) is non-resizable

			+ samples to display will be skipped (details are lost)


	Plotter

		Upper : Left Channel
		Lower : Right Channel

		Input

			click
				set position

			drag
				select area

			Shift
				move data

			Shift + Cmd
				move Left or Right

				+ apply a channel under cursor

			Backspace
				delete

		Menu

			Undo
				undo

			Cut
			Copy
			Paste
			Overwrite/Mix
				edit

			Mute
				set zero

			Copy Left to Right
			Copy Right to Left
				make monaural compatible data

			Fade-In
			Fade-Out
				fade effect

			Smoother
				Savitzky-Golay Filter


	Icon Buttons

		Mute
			set zero

		Mix
			mix on/off

			+ drop and effects behavior will change

		Stop
			stop playback

		Play
			playback

		Equalizer
			open a graphic equalizer window

		Resizer
			resize whole data

		Formatter / Save
			save as specified format

	Slider

		L/R
			normalizer
